Easy to filter - create expressions based on properties of the piped objects

# PowerShell:
> Get-Process `
| Where-Object {$PSItem.MainWindowTitle.Contains("PowerShell") }
 

//C#
GetProcess()
.Where(psItem => psItem.MainWindowTitle.Contains("PowerShell"));
# PowerShell - short form:
> Get-Process `
| ? {$_.MainWindowTitle.Contains("PowerShell") }

PowerShell piping feels similar to C# LINQ extension methods :-)

Prefer short-form for interactive, long-form for scripts

  • VS Code PowerShell extension helps

# ... or embrace piping
> Get-Process notepad `
| ? {$_.MainWindowTitle.Contains("foo.txt") }`
| Stop-Process
